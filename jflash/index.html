这是干虾米的插件，不用解释，直接看Demo吧！

多想想还有哪些Css属性可以做出想象的效果！

Demo：这些文件将表现插件的效果！

红色闪烁 背景闪烁 组合了颜色与背景闪烁 字体大小闪烁 内边距造成的横向震动

And u can think more。。。

调用的代码：
    var css = {
        color: {color: "#e00"}, //红色闪烁的css属性
        background: {backgroundColor: "#080"}, //背景闪烁的css属性, 如果属性名称为background，则该元素事先需要先css定义好
        group: {color: "#e00", background: "#080"}, //组合了颜色与背景闪烁
        font: {fontSize: "20px"}, //修改字体大小
        padding: {paddingLeft: "10px"} //内边距造成的横向震动
    };
    
    $("#trigger a").click(function(){
        var key = $(this).attr("name");
        $("#jFlash").jFlash({
            css: css[key] 
        });
    });

    /*
    //单独完整的调用是这样子的：（因为默认参数有时候是够用的，就不必再次设置了）
        $("#jFlash").jFlash({
