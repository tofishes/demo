本主题基于mg12(http://www.neoease.com/)的iNove（http://wordpress.org/extend/themes/inove）源代码进行开发。



主题简介
Poetry采用绿色为主色调，两栏结构，兼容IE6、IE7、IE8、FIREFOX等主流浏览器。
在载入过程方面，优先载入文章区域，提升用户体验。
在SEO方面，对文章摘要做了优化处理，便于搜索引擎的收录。

安装方法
1. 上传主题文件夹到wp-content/themes/目录
2. 进入管理界面点击外观-主题
3. 选择Poetry模板

备注

Description
This theme was designed and built by cuikai.
2 columns treats the theme poetically. Tested in Firefox, IE6 and IE7,SEO friendly.

Install
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to Admin WP and select Design.
3. Select Poetry theme.
